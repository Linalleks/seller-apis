# Что умеет seller.py

1. Получает артикулы товаров магазина Озон.
2. Получает остатки товаров с сайта timeworld.ru.
3. Обновляет остатки товаров магазина Озон по данным остатков с сайта timeworld.ru по следующим правилам:
   - если товар с сайта timeworld.ru есть в магазине Озон, то количество записывается по условиям:
     - если на сайте количество = ">10", то на Озон = 100,
     - если на сайте  количество = "1", на Озон = 0,
     - иначе, на Озон = количество с сайта.
   - для товаров магазина Озон, которых нет на сайте timeworld.ru, записывается количество = 0.
5. Обновляет цены товаров магазина Озон, которые есть на сайте timeworld.ru, в соответствии с ценами с сайта timeworld.ru.

# Что умеет market.py

Выполняет все следующие действия отдельно для товаров, продающихся по схеме FBS и продающихся по схеме DBS на Яндекс маркете:
1. Получает артикулы товаров магазина Яндекс маркета.
2. Получает остатки товаров с сайта timeworld.ru.
3. Обновляет остатки товаров магазина Яндекс маркета по данным остатков с сайта timeworld.ru по следующим правилам:
   - если товар с сайта timeworld.ru есть у продавца на Озон, то количество записывается по условиям:
     - если на сайте количество = ">10", то на Яндекс маркете = 100,
     - если на сайте  количество = "1", на Яндекс маркете = 0,
     - иначе, на Яндекс маркете = количество с сайта.
   - для товаров магазина Яндекс маркета, которых нет на сайте timeworld.ru, записывается количество = 0.
5. Обновляет цены товаров магазина Яндекс маркета, которые есть на сайте timeworld.ru, в соответствии с ценами с сайта timeworld.ru.
