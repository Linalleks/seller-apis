# Что умеет seller.py

1. Получает артикулы товаров из магазина клиента Озон.
2. Получает остатки часов с сайта casio.

3. Обновляет остатки часов в магазине клиента Озон по данным остатков часов с сайта casio по следующим правилам:
   - если часы с сайта casio есть на Озон:
     - если на сайте casio количество = ">10", то на Озон установить - 100,
     - если на сайте casio количество = "1", на Озон установить - 0,
     - иначе, на Озон установить количество с сайта casio.
   - если часов с сайта casio нет на озон, то добавить новые часы в остатки на Озон с количеством - 0.

5. Меняет цены часов в магазине клиента Озон на цены этих часов с сайта casio.
